### visualization-platform

### 项目介绍

一个前台基于 `Echarts` 和 `Vue.js` ，后台基于 `Koa` 的可视化大屏项目

### 文件结构

koa_server 服务器
dist 客户端
src 客户端的 Vue 工程文件

### 使用方法

1. `git clone`
2. `npm i`
3. `npm run serve`

### 技术栈

1. `Vue.js`
2. `Echarts`
3. `Koa`
4. `Vue-router`
5. `Vuex`

### 项目功能

- [x] 6 种图表的展示
- [x] 各图表的放大缩小功能
- [x] 一键换肤
- [ ] `websocket` 各屏联动效果
